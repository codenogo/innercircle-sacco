# Plan 04: Add unit tests for new auth components (JwtService, AuthService, MeController, UserAdminController)

## Goal
Add unit tests for new auth components (JwtService, AuthService, MeController, UserAdminController)

## Tasks

### Task 1: Unit tests for JwtService
**Files:** `sacco-security/src/test/java/com/innercircle/sacco/security/service/JwtServiceTest.java`
**Action:**
Create JwtServiceTest with @ExtendWith(MockitoExtension.class). Test: generateAccessToken returns valid JWT with correct claims (sub, userId, email, roles, authorities, iss, exp); generateRefreshToken returns 64-char hex string; getJwtDecoder can decode tokens generated by generateAccessToken; token with expired time is rejected; ephemeral key fallback works when no RSA config provided.

**Verify:**
```bash
mvn -pl sacco-security test -Dtest=JwtServiceTest -q
```

**Done when:** [Observable outcome]

### Task 2: Unit tests for AuthService
**Files:** `sacco-security/src/test/java/com/innercircle/sacco/security/service/AuthServiceTest.java`
**Action:**
Create AuthServiceTest with @ExtendWith(MockitoExtension.class). Mock: UserAccountRepository, RefreshTokenRepository, JwtService, PasswordEncoder. Test: authenticate success returns tokens; authenticate with wrong password throws BadCredentialsException; authenticate with disabled account throws DisabledException; authenticate with locked account throws LockedException; refreshAccessToken success rotates token; refreshAccessToken with expired/revoked token throws; refreshAccessToken with unknown token throws.

**Verify:**
```bash
mvn -pl sacco-security test -Dtest=AuthServiceTest -q
```

**Done when:** [Observable outcome]

### Task 3: Unit tests for MeController and UserAdminController
**Files:** `sacco-security/src/test/java/com/innercircle/sacco/security/controller/MeControllerTest.java`, `sacco-security/src/test/java/com/innercircle/sacco/security/controller/UserAdminControllerTest.java`
**Action:**
Create MeControllerTest with @ExtendWith(MockitoExtension.class). Mock UserAccountRepository, MemberRepository. Test: getCurrentUser returns user with member info; getCurrentUser returns user without member; getCurrentUser with unknown user returns 404. Test: createUser returns 201 with user details; createUser triggers password reset email when flag set; triggerPasswordReset calls service correctly.
